HTML Basics

  • Bold tag - defined using "b"
  • Italics - defined using "em" (emphasis)
  • Bold tag - defined using "b"
  • Computers are stupid - they cannot deal with ambiguity and only interpret language literally. Because of this, they are sensitive to typos and other small mistakes.
  • Link (anchor) tag - includes href attribue with link in quotes
  • Image tag - requires src (source) and alt (alternate text) attributes, no closing tag
  • White space is mostly ignored by HTML
  • br tag is used to create a line break
  • paragraph (p) tag is a block element that also creates line breaks
  • Inline vs block elements - inline elements don't create line breaks or box (block), while block elements do
  • Container elements: div (block) and span(inline)
  • HTML document structure:
    1. doctype declaration informs browser what kind of document this
    2. html tags surround document
    3. head contains metadata, title, links to CSS and Javascript
    4. body cotains renderable contents

Summary
World Wide Web:
composed of HTML files and utilizes HTTP for interactions between servers and clients

tell a web browser how to display content on a web page

provide additional information to the browser
